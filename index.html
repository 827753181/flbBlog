react fiber结构学习小计

发表于2020-03-03更新于2021-03-18

所谓 react fiber 结构，个人理解是一个双向的链表控制的一些自建的虚拟 dom。
每个虚拟 dom（即 fiber 节点），内部有一个 stateNode 指向真实的 dom 节点，借着这些虚拟 dom，控制的一个树状双向链表结构。
